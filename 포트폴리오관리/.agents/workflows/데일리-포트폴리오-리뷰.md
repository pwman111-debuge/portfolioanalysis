# 데일리 포트폴리오 리뷰 (Daily Portfolio Review)

## 실행 조건
마스터 제네시스 워크플로우에서 포트폴리오 리뷰 단계에 도달하면 **항상** 실행.
캡처 이미지 첨부 여부와 무관하게 실행되며, 데이터 수집 방법만 아래 1단계에서 분기한다.

## 투자 목표 (항상 기억)
- **연 수익률 목표: 25%**
- 한국주식 월 220만원 (중기 70%=154만원, 장기 30%=66만원)
- 미국주식 월 220만원 (QQQM/SCHD/VZ/JEPI/VHT 기계적 DCA - 리뷰 불필요)

---

## 0단계: 포트폴리오 대장 읽기 (가장 먼저 실행)

```
읽을 파일: 포트폴리오관리/포트폴리오-대장.md
확인 항목:
  - 현재 등록된 보유 종목 전체 목록
  - 각 종목의 분류(중기/장기), 섹터구분(일반/대세)
  - 목표가, 1차/2차 손절가, 현재 상태
  - 이달 잔여 예산
→ 이후 모든 판단의 기준값으로 사용
```

---

## 1단계: 계좌 데이터 수집

### 1-0. 수집 방법 선택 (자동 분기)

```
캡처 이미지가 대화에 첨부되어 있으면 → [방법 A] 이미지 분석
캡처 이미지가 없으면               → [방법 B] playwright MCP 자동 크롤링 (기본값)
```

---

### [방법 A] 캡처 이미지 분석

이미지가 첨부된 경우 아래 항목을 추출한다.

**[계좌 전체 캡처]**
- 총 평가금액
- 총 수익률 (%)
- 총 수익금 (원)
- 현금 비중 (%)
- 한국주식 / 미국주식 비중

**[한국주식 보유 현황 캡처]**
각 보유 종목별:
- 종목명 / 종목코드
- 보유 수량
- 평균 매수가
- 현재가
- 평가손익 (금액 + %)

**[미국주식 보유 현황 캡처]**
- DCA 진행 현황 확인 (QQQM/SCHD/VZ/JEPI/VHT)
- 환율 영향 체크
- 이상 종목 여부만 확인 (DCA는 개입 없음)

**[당일 매수 내역 캡처 - 있을 경우]**
- 종목명 / 매수가 / 수량 / 총금액

**[당일 매도 내역 캡처 - 있을 경우]**
- 종목명 / 매도가 / 수량 / 실현손익

---

### [방법 B] playwright MCP 자동 크롤링 (기본값)

캡처 이미지가 없을 때 **항상 이 방법을 사용한다.**
토스증권 URL은 **https://tossinvest.com** 으로 고정한다 (변경 금지).

**크롤링 순서:**

```
Step 1. 메인 페이지 접속
  → mcp__playwright__browser_navigate: https://tossinvest.com

Step 2. 스크린샷으로 로그인 상태 확인
  → mcp__playwright__browser_take_screenshot

Step 3. 투자 포트폴리오 페이지 이동
  → mcp__playwright__browser_navigate: https://tossinvest.com/investment-portfolio

Step 4. 페이지 스냅샷 또는 스크린샷으로 계좌 현황 파악
  → mcp__playwright__browser_snapshot (또는 browser_take_screenshot)
  → 추출 항목: 총 평가금액, 총 수익금(%), 일간 손익, 원화 예수금

Step 5. 국내주식 전체 목록 확인
  → 종목명, 평균단가, 현재가, 수익률, 총손익, 일간 손익 모두 추출
  → 대장 등록 종목 수와 대조하여 누락 없는지 확인

Step 6. 해외주식 현황 확인
  → DCA 종목(QQQM/SCHD/VZ/JEPI/VHT) 현황 및 이상 여부만 확인

Step 7. 거래 내역 확인 (당일 매수/매도 유무)
  → mcp__playwright__browser_navigate: https://tossinvest.com/transaction-history
  → 당일 국내주식 매수/매도 내역 추출 (없으면 "없음" 기록)

Step 8. 배당금 입금 여부 확인
  → 거래 내역에서 당일 배당금 항목 확인
  → 있으면 종목명 + 금액 기록 (대장 배당 이력에 반영)
```

**크롤링 실패 시 처리:**
```
접속 실패 또는 로그인 필요 화면 → 스크린샷 출력 후 황원장님에게 알림
  "playwright MCP로 tossinvest.com 접근 실패. 캡처 이미지를 첨부해주시면 분석 진행합니다."
```

## 1-2단계: 미등록 종목 분류 확정 (필수 선행)

1단계에서 수집한(이미지 또는 playwright MCP) 한국주식 보유 종목을 대장과 대조한다.

```
대조 결과:
  ✅ 대장에 있는 종목 → 기존 분류 그대로 사용
  ❌ 대장에 없는 종목 → 아래 절차 즉시 실행 (행동 지침 판단 전에 반드시 완료)
```

### 미등록 종목 발견 시 처리 절차

**Step 1. 미등록 종목 목록 출력**
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
⚠️ 포트폴리오 대장에 미등록된 종목이 있습니다.
행동 지침 판단 전에 분류를 먼저 확정해야 합니다.

[미등록 종목]
1. [종목명] - 평균매수가: [금액] / 현재 수익률: [%]
2. [종목명] - 평균매수가: [금액] / 현재 수익률: [%]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**Step 2. Claude의 자동 분류 제안**

각 미등록 종목에 대해 섹터 지식 기반으로 분류를 먼저 제안한다:
```
[종목명] 분류 제안:
- 보유 성격: 중기 / 장기  ← 매수 시점, 수익률, 시황 참고하여 제안
- 섹터구분: 대세 / 일반   ← 종목의 산업 섹터 기반 제안
- 목표가 제안: [금액]      ← 중기/장기 추천 리포트 참고
- 손절가 제안: 1차 [금액] / 2차 [금액]

→ 황원장님 확인 후 확정해주세요.
```

**Step 3. 사용자 확인 후 대장 업데이트**
```
황원장님이 분류를 확정하면:
→ 포트폴리오-대장.md 보유 종목 테이블에 즉시 추가
→ 이달 투입 금액 업데이트
→ 이후 행동 지침 판단 진행
```

## 3단계: 보유 종목별 행동 지침 판단

각 한국주식 보유 종목에 대해 아래 순서로 판단한다.

### 3-1. 보유 종목 분류 (2가지 축)

**축 1 - 보유 성격:**
```
중기: 3~6개월 목표 포지션
장기: 1~3년 목표 포지션
→ 포트폴리오 대장 또는 이전 리뷰 기록 기준으로 판단
```

**축 2 - 섹터 분류:**
```
→ 포트폴리오-대장.md의 "현재 지정 대세 섹터" 섹션을 읽어 기준으로 사용한다.
  (하드코딩 금지 — 대장이 업데이트될 때마다 자동으로 최신 기준 반영)

일반 섹터: 대장의 현재 지정 대세 섹터에 미해당 종목 전체
```

→ 위 두 축 조합으로 4가지 기준 중 하나 적용:
- **[A] 중기 × 일반섹터**
- **[B] 중기 × 대세섹터**
- **[C] 장기 × 일반섹터**
- **[D] 장기 × 대세섹터**

---

### 3-2. 익절 판단 기준

**[A] 중기 × 일반섹터**
```
1차 익절: 수익률 +15% 도달 → 보유 수량의 1/3 매도
2차 익절: 수익률 +25% 도달 → 보유 수량의 1/3 매도
최종 익절: 목표가 100% 도달 → 잔량 전량 매도
```

**[B] 중기 × 대세섹터**
```
1차 익절: 수익률 +20% 도달 → 보유 수량의 1/4 매도
2차 익절: 수익률 +40% 도달 → 보유 수량의 1/4 매도
3차 익절: 수익률 +60% 도달 → 보유 수량의 1/4 매도
잔량 1/4: thesis 훼손 시까지 무제한 보유
```

**[C] 장기 × 일반섹터**
```
1차 익절: 수익률 +30% 도달 → 보유 수량의 1/3 매도
2차 익절: 수익률 +50% 도달 → 보유 수량의 1/3 매도
최종 익절: thesis 훼손 확인 시 → 잔량 전량 매도
```

**[D] 장기 × 대세섹터**
```
1차 익절: 수익률 +50%  도달 → 보유 수량의 1/4 매도
2차 익절: 수익률 +100% 도달 → 보유 수량의 1/4 매도
3차 익절: 수익률 +150% 도달 → 보유 수량의 1/4 매도
잔량 1/4: 섹터 구조 붕괴 시까지 보유
```

---

### 3-3. 손절 판단 기준

**[A] 중기 × 일반섹터**
```
1차 경고: 손실률 -8%  → ⚠️ Claude와 재확인 후 결정
2차 손절: 손실률 -12% → 🚨 전량 매도 강력 권고
```

**[B] 중기 × 대세섹터**
```
1차 경고: 손실률 -10% → ⚠️ Claude와 재확인 후 결정
2차 손절: 손실률 -15% → 🚨 전량 매도 강력 권고
```

**[C] 장기 × 일반섹터**
```
1차 경고: 손실률 -15% → ⚠️ Claude와 재확인 후 결정
2차 손절: 손실률 -20% → 🚨 전량 매도 강력 권고
```

**[D] 장기 × 대세섹터**
```
1차 경고: 손실률 -20% → ⚠️ Claude와 재확인 후 결정
2차 손절: 손실률 -25% → 🚨 전량 매도 강력 권고
```

> 손절 1차 경고: 반드시 황원장님과 대화 후 최종 결정
> 손절 2차: 강력 권고이나 최종 결정권은 황원장님

---

### 3-4. 보유 판단 기준 (3가지 모두 충족 시)
```
✅ 시황 중립 이상
✅ 기관/외국인 수급 이탈 없음
✅ 투자 thesis 훼손 없음
→ 위 익절/손절 기준에 해당하지 않으면 보유 유지
```

---

### 3-5. 즉시 매도 조건 (수익률/섹터 무관)
```
🚨 대규모 유상증자 공시
🚨 횡령/배임 등 지배구조 리스크
🚨 핵심 사업 철수 or 대형 계약 해지
🚨 시황 "하락추세" 전환 + 수급 동반 이탈
```

### 3-6. 대세 섹터 즉시 강등 조건

아래 이벤트 발생 시 월간 리뷰를 기다리지 않고 **당일 즉시** 처리한다.

```
즉시 강등 트리거 (아래 중 1개라도 해당):
🔻 글로벌 선도 기업 실적 쇼크 (엔비디아, TSMC 등 핵심 기업 어닝쇼크)
🔻 핵심 정책 폐기 or 예산 대폭 삭감 공식 발표
🔻 섹터 전체 외국인/기관 3주 연속 동반 이탈
🔻 대체 기술 등장으로 섹터 thesis 붕괴
```

즉시 강등 확정 시 아래 2가지를 모두 실행한다 (6단계에서 처리):
1. **포트폴리오-대장.md "현재 지정 대세 섹터"** → 해당 섹터 항목 삭제
2. **보유 종목 테이블** → 해당 섹터 보유 종목의 섹터구분 대세→일반 변경 + 목표가·손절가 재산정

## 4단계: 신규 매수 검토

### 4-1. 월 예산 및 투입 가이드라인

**월 예산 배분 원칙**
```
한국주식 월 예산: 220만원
  ├── 중기 70% = 154만원 (동시 최대 3종목)
  │     종목당 평균: 약 51만원
  └── 장기 30% =  66만원 (동시 최대 2종목)
        종목당 평균: 약 33만원
```

**시황 판단 출처**
```
공격/중립/방어  → 단기유망종목 리포트 Step 1 결과 (매일 생성)
시장 국면       → 시황분석 리포트 결과 (참고용)

두 지표 매핑:
  단기 공격  ←→  시황 상승추세
  단기 중립  ←→  시황 조정 중
  단기 방어  ←→  시황 하락추세
  
→ 투입 강도 판단은 단기유망종목의 공격/중립/방어 기준으로 결정
```

**시황별 투입 강도**
```
공격            → 월 예산 100% 소진 가능 (적극 투입)
중립            → 월 예산 50% 이하 투입 (선별 투입)
방어            → 투입 전면 보류 (현금 유지)
방어 심화(3일↑) → 투입 금지 + 기존 포지션 축소 검토
```

**분할 매수 원칙 (종목당)**
```
1차 진입: 해당 종목 목표 금액의 60%
          → 시황 + 수급 신호 확인 시 즉시 진입

2차 추가: 나머지 40%
          → 중기: 1차 매수 후 조정 시 or 추가 확신 시
          → 장기: 분기 실적 확인 후 추가 매수
```

### 4-2. 현금 비중 및 잔여 예산 확인
```
대장에서 이달 투입 금액 확인
잔여 예산 = 220만원 - 이달 투입 금액
  └── 잔여 예산이 0 이하면 → 이달 신규 매수 불가
  └── 잔여 예산이 있어도 시황 방어/하락 시 → 투입 보류
```

### 4-3. 신규 매수 후보 제안 (기본 예산)
```
시황 공격/중립 → 기본 예산 투입 진행
시황 방어       → 기본 예산도 투입 속도 늦춤 (월말까지 천천히)

[중기 후보 - 잔여 예산의 70% 범위]
  - 대장 현재 중기 종목 수 확인
  - 3종목 미만일 때만 신규 진입 제안
  - 중기 추천 리스트 우선순위 상위 종목 제안
  - 제안 금액: 종목당 목표금액의 60% (1차 진입)

[장기 후보 - 잔여 예산의 30% 범위]
  - 대장 현재 장기 종목 수 확인
  - 2종목 미만일 때만 신규 진입 제안
  - 장기 추천 리스트 종목 제안
  - 제안 금액: 종목당 목표금액의 60% (1차 진입)
```

### 4-4. 추가 재원 운용 (익절/손절 회수금)

> 월 220만원 기본 예산과 별도로 관리되는 추가 투자 재원

**재원 확인**
```
대장의 [추가 재원 현황] 섹션에서 확인:
  - 이달 익절 회수금 누계
  - 이달 손절 회수금 누계
  - 추가 재원 잔액
```

**시황별 운용 기준**
```
공격 시황:
  익절금 → 100% 즉시 재투자 가능
  손절금 → 3거래일 경과 후 재투자 가능 (감정 매매 방지)

중립 시황:
  익절금 → 50% 재투자 / 50% 현금 보유
  손절금 → 전액 현금 보유 (공격 전환 시 재투자)

방어 시황:
  익절금 → 전액 현금 보유
  손절금 → 전액 현금 보유
  → 추가 재원 집행 전면 보류
```

**추가 재원 재투자 우선순위**
```
1순위: 기존 포트 종목 중 2차 분할 매수 미완료 종목
       → 해당 종목의 나머지 40% 매수에 우선 활용

2순위: 중기/장기 추천 리스트 신규 종목
       → 기존 포트 추가 매수 불필요 시 신규 진입

3순위: 현금 보유
       → 1,2순위 해당 없거나 방어 시황 시
```


## 6단계: 포트폴리오 대장 업데이트

리뷰 리포트 저장 후 대장을 최신 상태로 유지한다.

```
업데이트 항목:

1. 당일 매수 내역이 있을 경우
   → 보유 종목 테이블에 신규 행 추가
   → 이달 투입 금액 업데이트

2. 당일 매도(익절/손절) 내역이 있을 경우
   → 전량 매도: 해당 행 삭제 + 매도 이력에 기록
   → 분할 매도: 보유수량 및 상태 업데이트 + 매도 이력에 기록
   → 추가 재원 현황 업데이트:
      익절 시: 이달 익절 회수금 누계 + 추가 재원 총액 갱신
      손절 시: 이달 손절 회수금 누계 + 추가 재원 총액 갱신
      손절 날짜 기록 (3거래일 쿨다운 계산용)

3. 대세 섹터 즉시 강등(3-6)이 발생한 경우
   → 포트폴리오-대장.md "현재 지정 대세 섹터" 목록에서 해당 섹터 삭제
   → 해당 섹터 보유 종목 전체: 섹터구분 대세 → 일반으로 수정
   → 변경된 분류 기준으로 목표가·손절가 재산정 후 업데이트
   → 당일 특이사항 섹션에 강등 내용 기재

4. 변경사항 없을 경우
   → 대장 상단 "마지막 업데이트" 날짜만 갱신

5. 당일 특이사항 섹션 업데이트 (매번 반드시 실행)
   → 대장 맨 아래 "## 당일 특이사항" 섹션의 이전 내용을 전부 삭제하고 오늘 내용으로 덮어쓴다
   → 예산 초과, 손절 쿨다운 시작, 대세 섹터 변경 등 오늘 발생한 특이사항만 기재
   → 특이사항 없으면 "(없음)"으로 기재
   → 절대 누적하지 않는다 (이전 날짜 내용이 남아있으면 삭제)
```

## 주의사항
- 미국주식(QQQM/SCHD/VZ/JEPI/VHT)은 DCA 전략이므로 개입 금지
- 1차 손절 경고는 반드시 황원장님과 대화 후 최종 결정
- 시황 "방어" 또는 "하락추세" 전환 시 신규 매수 제안 금지
- 대세 섹터 판단은 오늘 시황분석 결과와 반드시 교차 확인
- 익절/손절 최종 결정권은 항상 황원장님에게 있음
