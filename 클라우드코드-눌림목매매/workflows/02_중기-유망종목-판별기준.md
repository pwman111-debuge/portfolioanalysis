# 중기 유망종목 판별 기준 (Medium-term / 3~6개월)

> **핵심 철학**: 펀더멘털 + 수급이 동시에 유효한 종목을 스코어카드(20점 만점)로 정량 평가, 최종 3종목 확정

---

## Step 1. 시장 국면 및 주도 섹터 선정

### 1-1. 시장 국면 판단 (4가지 중 1개 확정)
- `상승추세 초입` / `상승추세 지속` / `조정 중` / `하락추세`
- KOSPI/KOSDAQ 20일·60일 이평선 위/아래 여부 + 최근 5거래일 방향성
- 외국인·기관 **최근 2주** 자금 유입 방향

### 1-2. 섹터 선별 (2~3개 압축)
등락률 상위 5개 업종 중 아래 조건으로 필터:
- **거래대금 증가 + 외국인/기관 동반 유입** (진성 모멘텀)
- 정책·수출·기술 트리거 존재 (단순 테마성 제외)
- 분기 실적 시즌과 연동
- 글로벌 동종 업종과 방향 일치

### 1-3. 보조 확인
- 테마 모멘텀 교차 검증 (선별 섹터와 중첩 여부)
- 증권사 리포트 트렌드 — 최근 1~2주 **목표주가 상향 + BUY 신규** 집중 섹터
- 글로벌 변수: 달러/원 환율, 미 증시 주요 섹터 (보강용)

---

## Step 2. 펀더멘털 스크리닝 (1차 정량 필터)

### 2-1. 통과 필수 조건 (Pass/Fail)
- [ ] 시가총액 **500억 이상**
- [ ] 증권사 투자의견 커버리지 **2개사 이상**
- [ ] 평균 목표주가 대비 현재가 **괴리율 +15% 이상**
- [ ] 최근 분기 영업이익 **흑자**

### 2-2. 제외 조건 (권리변경 / 리스크)
- 최근 1주일 내 **액면분할 / 유·무상증자 / 거래정지 / 관리종목 지정** 공시

### 2-3. 펀더멘털 심층 분석 수치 기준

**① 실적 성장성**
- 최근 4분기 누적 영업이익 **YoY +10% 이상**
- 다음 분기 컨센서스 **상향 추세**
- 어닝 서프라이즈 (+5% 초과) 여부

**② 재무 건전성**
- ROE **10% 이상**
- 부채비율 **150% 이하**
- 유동비율 **100% 이상**

**③ 밸류에이션**
- PER: 섹터 평균 대비 할인 or **PEG < 1**
- PBR: 역사적 밴드 하단 위치
- 증권사 목표주가 괴리율 + 최근 3개월 내 리포트 발행 확인

### 2-4. 기술적 위치
- 현재가가 **20일선·60일선 위**
- 주요 지지 구간(전 고점, 120일선) 근처 → 손절가 근거
- **제외**: RSI 70 초과 과열 or 전고점 대비 -40% 이상 하락 중

### 2-5. 촉매(Catalyst) 존재 여부
- 대규모 수주 / 신규 사업 / 설비 증설
- 실적 발표 예정 + 컨센서스 상회 가능성
- 정부 정책 수혜 (예산, 인허가)

### 2-6. Step 2 스코어카드 (10점)
| 항목 | 배점 |
|------|------|
| 실적 성장 YoY +10% 이상 | 2점 |
| 재무 건전성 통과 | 2점 |
| 밸류에이션 저평가 | 2점 |
| 기술적 위치 양호 (이평선 위) | 1점 |
| 명확한 촉매 존재 | 2점 |
| 증권사 커버리지 3개사 이상 | 1점 |

- **8점 이상** → Step 3 우선 분석
- **6~7점** → Step 3 후순위 분석
- **5점 이하** → 제외

---

## Step 3. 수급 및 투자 심리 분석

### 3-1. 기관 수급 (3개 기간 교차 검증)
| 기간 | 확인 포인트 |
|------|------------|
| 최근 5일 | 단기 집중 매수 (급등 직전 진입 신호) |
| 최근 20일 | 중기 누적 순매수 방향 |
| 최근 60일 | 장기 관심 (저점 분할 매수 패턴) |

- **강한 신호**: 3개 기간 모두 순매수
- **주의 신호**: 5일만 매수, 60일은 매도 → 신뢰도 낮음
- 프로그램 비차익 매수 증가 시 ETF 편입 기대

### 3-2. 외국인 수급
- 최근 1개월 지분율 증가 여부
- 순매도→순매수 **전환 시점 최근 2주 이내**
- 한도소진율 80% 이상 → 리스크 기록
- **쌍끌이(기관+외국인 동일 기간 순매수) → 최우선 선정**

### 3-3. 공매도 / 대차잔고
- 공매도 비율 일평균 거래대금 대비 **5% 초과 → 감점**
- 대차잔고 최근 2주 감소 추세 → 숏커버링 기대 (긍정)
- 공매도 과열 종목 지정 시 중기 리스크 기록

### 3-4. 개인 투자자 심리 (과열 필터)
**과열 경고 (선정 보류)**
- 종목토론실 "무조건 오른다" 집단 낙관
- 신용잔고 최근 1개월 **+30% 이상 급증**
- 이미 핫이슈로 과도 소비된 상태

**저평가 심리 (긍정 — 매수 적기)**
- 토론실 조용 or 실망 분위기
- 신용잔고 안정 or 소폭 감소
- 기관만 조용히 매수 중

### 3-5. Step 3 스코어카드 (10점)
| 항목 | 배점 |
|------|------|
| 기관 3개 기간 모두 순매수 | 3점 |
| 외국인 순매수 전환 (최근 2주) | 2점 |
| 쌍끌이 매수 확인 | 2점 |
| 공매도 낮음 + 대차잔고 감소 | 1점 |
| 개인 심리 과열 없음 | 1점 |
| 신용잔고 안정적 | 1점 |

## 최종 선정 기준 (Step 2 + Step 3 합산)

| 합산 점수 | 판정 |
|----------|------|
| **14점 이상** | 최종 선정 0순위 |
| 10~13점 | 3슬롯 여유 시 포함 |
| 9점 이하 | 이번 회차 제외 |

→ **최종 3종목 확정** (섹터 중복 최소화)

## 가격 전략 필수 항목
```
현재가:      X,XXX원 (확정 시각 명시)
목표가:      X,XXX원 (+XX%)   ← 컨센서스 or 목표 PER 기반
1차 손절가:  X,XXX원 (-XX%)   ← 주요 지지선 or 60일선 하단 (-10~15%)
2차 손절가:  X,XXX원 (-XX%)   ← thesis 훼손 수준 (-20%)
R/R 비율:    X : 1
권장 보유 기간: 3~6개월
```

## 포트폴리오 관점 검토
- **섹터 중복 금지**: 3종목이 동일 섹터에 몰리면 차상위 후보로 교체
- **시가총액 혼합**: 대형주 1 + 중형주 1~2
- **모멘텀 타입 혼합**: 실적 성장형 + 수급 주도형 + 정책 수혜형 중 2개 이상

## 최종 단계: 눌림목 타점 필터 + 대세 태깅

### Step A. 눌림목 체크리스트 통과 여부
`00_눌림목-정의.md` §6 적용 — 중기는 **20일선·60일선 눌림**을 주 타점으로.
- 통과 → `eligible: 진입 가능`
- 실패 → `eligible: 관망 대기`

### Step B. 대세섹터 여부 판정
`04_대세섹터-판별기준.md` §1 기준:
- 해당 시 → **중기 × 대세(+)**
- 미해당 시 → **중기 × 대세(−)**

### Step C. 최종 출력 포맷
```yaml
bucket: 중기×대세(+)   # or 중기×대세(−)
eligible: 진입 가능    # or 관망 대기
term: medium
horizon_months: 3~6
entry: X,XXX원
target: X,XXX원 (+XX%)
stop_loss_1: X,XXX원 (-XX%)
stop_loss_2: X,XXX원 (-XX%)
score: 2단계+3단계 합산 (최대 20점)
pullback_zone: 20일선 근접 / 60일선 눌림 / 전고점 지지 / 등
catalyst: [핵심 촉매 이벤트]
```
