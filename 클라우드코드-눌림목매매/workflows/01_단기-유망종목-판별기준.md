# 단기 유망종목 판별 기준 (Short-term)

> **보유 기간**: 기본 **5~10거래일 (1~2주)** / 상한 **3주**
> **핵심 철학**: 수급 + 기술적 타점이 동시에 맞는 종목을 당일 시황에 맞춰 3종목(방어 시 2종목) 압축

**기간 근거**
- 5·20일 이평선 기반 전략 → 신호 유효 구간 1~2주
- R:R 2:1(+10%/-5%) 1차 목표 달성 평균 시간 5~10거래일
- 3주 초과 시 수급 이탈·모멘텀 소멸 가능성 높음
- 매일 신규 리포트 발행 → 회전율 중시

---

## Step 1. 시장 환경 판단 (선결 조건)

### 1-1. 필수 지표 수집
| 지표 | 확인 사항 |
| :--- | :--- |
| KOSPI / KOSDAQ | 지수·등락률·거래대금 |
| USD/KRW | 환율 흐름 |
| 수급 주체 | 외국인·기관 순매수/순매도 |

### 1-2. 시장 판단 (반드시 1개 확정)
| 판단 | 조건 | 액션 |
| :--- | :--- | :--- |
| **공격** | 외인+기관 쌍끌이, 거래대금 20조↑ | 종목 3개 발굴 |
| **중립** | 수급 혼조, 거래대금 보통 | 종목 3개 발굴 (비중 축소) |
| **방어** | 외인+기관 쌍매도, 지수 하락압력 | 종목 2개로 축소, 리스크 강화 |

---

## Step 2. 종목 스크리닝 (수급 + 기술)

### 2-1. 수급 스크리닝 — 후보군 5~7개 압축
1. **외국인 순매수 상위** KOSPI/KOSDAQ 각 10종목
2. **기관 순매수 상위** KOSPI 10종목 — **외국인 리스트와 겹치면 최우선**
3. **거래량 급증**: 전일比 2배 이상 + 상승 마감
4. **주도 테마**: 당일 상승률 상위 테마 1~2개 내 대장주

### 2-2. 기술적 타점 검증 — 3~5개로 압축
- **현재가 / 최근 5일 흐름** (상승·하락·횡보)
- **이동평균선 정배열** (5일·20일·60일) → **정배열만 통과**
- **RSI 과열 여부**: 급등 연속 3일↑ = 제외
- **거래량 패턴**: 음봉 감소 → 양봉 폭발 확인

---

## Step 3. 리스크 필터링 (즉시 제외 조건)

### 3-1. 공시 리스크 — 하나라도 해당 시 탈락
- [ ] 최근 5거래일 내 **유상증자·전환사채(CB)·주식 담보대출** 공시
- [ ] **투자경고·투자위험 종목** 지정 이력
- [ ] 최대주주·임원의 **대량 지분 매도** 공시
- [ ] 거래소 **조회 공시 요구** 수신

### 3-2. 모멘텀 유효성
- [ ] 상승 이유가 **뉴스/공시로 뒷받침** — 근거 없는 급등은 세력 개입 의심 → 제외
- [ ] **섹터 대장주 지위** — 2~3등주보다 대장주 우선

### 3-3. 최종 3종목 확정
- **R:R (목표/손절) 2:1 이상만 선정**
  - 예) +10% / -5% → R:R 2:1 통과
  - 예) +8% / -7% → R:R 1.14 → **제외**

## 리스크 가이드 (공통)
- 수급 이탈 시 즉시 비중 축소
- 시장 판단이 **방어**로 바뀌면 전량 관망
- R:R 2:1 미만 종목은 선정 금지

---

## 최종 단계: 눌림목 타점 필터 + 대세 태깅

### Step A. 눌림목 체크리스트 통과 여부
`00_눌림목-정의.md` §6 체크리스트를 모든 최종 후보에 적용:
- 통과 → `eligible: 진입 가능`
- 실패 → `eligible: 관망 대기` (타점 재형성 시 재평가)

### Step B. 대세섹터 여부 판정
`04_대세섹터-판별기준.md` §1 기준으로 판정:
- 해당 시 → **단기 × 대세(+)**
- 미해당 시 → **단기 × 대세(−)**

### Step C. 최종 출력 포맷
```yaml
bucket: 단기×대세(+)   # or 단기×대세(−)
eligible: 진입 가능    # or 관망 대기
term: short
horizon_days: 5~10
entry: X,XXX원
target: X,XXX원 (+XX%)
stop_loss: X,XXX원 (-XX%)
rr: 2:1
pullback_zone: 20일선 근접 / 5일선 눌림 / 등
```

## 보유 기간 관리 룰 ⭐

| 경과일 | 상태 | 조치 |
|--------|------|------|
| 1~10거래일 | 표준 보유 구간 | 목표가·손절가 기준 대응 |
| 11~15거래일 | 연장 구간 | 수급 유지 시만 보유, 과열 징후 시 청산 |
| **3주(15거래일) 초과** | **재평가 필수** | ① 펀더멘털 양호 → **중기로 재분류**<br>② thesis 약화 → **손절·익절 청산** |

> 단기 리포트는 매일 갱신되므로, 3주 넘도록 끌고 가는 것은 원 전략 이탈.
> 연속 2회(2주) 이상 동일 종목 재추천 시 중기 후보로 승격 검토.
