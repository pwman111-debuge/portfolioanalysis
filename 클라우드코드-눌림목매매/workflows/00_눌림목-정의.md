# 눌림목(Pullback) 정의

> **한 줄 정의**
> 상승 추세가 유효한 종목에서, 거래량·조정폭·기간 조건을 모두 충족하는 **일시적 조정 구간** — 재상승 타점으로 기능

---

## 1. 필수 선행 조건 — 상승 추세 확인 (4개 중 3개 이상)

| # | 조건 | 기준 |
|---|------|------|
| ① | 이평선 배열 | **20일선 > 60일선** + 상승 기울기 유지 |
| ② | 직전 상승폭 | 최근 20거래일 내 저점比 일정폭 이상 상승 (아래 표) |
| ③ | 고점 돌파 | 최근 20일 내 **신고가 경신** 이력 |
| ④ | 수급 유입 | 외국인 or 기관 직전 5~20일 **순매수** |

### 선행 상승폭 기준 (시총 규모별 차등)

| 시총 구분 | 기준 |
|----------|------|
| **대형주** (시총 3조↑) | +10% 이상 |
| **중형주** (5천억~3조) | +15% 이상 |
| **중소형주** (500억~5천억) | +20% 이상 |

> 3개 미만 충족 시 → **"눌림"이 아니라 추세 전환** 가능성 → 진입 금지

---

## 2. 눌림 자체의 조건 (3개 모두 충족)

### 2-1. 조정 폭

| 시총 구분 | 정상 조정 폭 |
|----------|------------|
| **대형주** | -5% ~ -10% |
| **중형주** | -7% ~ -12% |
| **중소형주** | -10% ~ -15% |

- **피보나치 되돌림 기준**: 직전 상승의 **38.2% ~ 61.8%** 구간
- 이 범위를 벗어나면 (작으면) 미성숙 / (크면) 추세 훼손

### 2-2. 조정 기간

| 구간 | 판정 |
|------|------|
| 3~10거래일 | ✅ 정상 눌림 |
| 11~15거래일 | ⚠️ 연장 구간 — 수급·거래량 재검증 필요 |
| **15거래일 초과** | ❌ 눌림 아님 (횡보 or 추세 전환) → 제외 |

### 2-3. 거래량 패턴

| 패턴 | 판정 |
|------|------|
| 조정 구간 **거래량 감소** (매물 소화) | ✅ 건전한 눌림 |
| 조정 구간 **거래량 평이** | 🟡 중립 (타 지표로 판단) |
| 조정 구간 **거래량 급증** | ❌ 투매·손절매 → 제외 |

---

## 3. 진입 타점 (3층 지지 구조)

### 주 타점: 20일선 눌림 ⭐
> 워크스페이스 표준 — 단기·중기 공통 주 진입 타점

| 유형 | 지지선 | 활용 기간 | 비중 |
|------|--------|---------|------|
| **5일선 눌림** | 5일 이평선 터치 | 단기 (1~3일 조정) | 보조 — 공격적 단기 |
| **20일선 눌림 ⭐** | 20일 이평선 터치/근접 | **단기·중기 공통** | **주 타점 (표준)** |
| **60일선 눌림** | 60일 이평선 터치 | 중기 깊은 조정 | 보조 — 방어적 진입 |
| **120일선 눌림** | 120일 이평선 | 장기 | 장기 2차 분할 타점 |
| **전 고점 지지** | 이전 돌파 박스권 상단 | 중·장기 | 추세 재확인 시 |

### 피보나치 되돌림 병행 확인
- **38.2%**: 강세 눌림 (공격적 진입)
- **50%**: 표준 눌림 (기본 진입)
- **61.8%**: 약세 눌림 (확인 후 진입)

---

## 4. 배제 조건 (하나라도 해당 시 제외)

```
❌ 이평선 역배열 (20일 < 60일)
❌ 최근 20일 고점(일봉 기준) 대비 -15% 초과 하락 (추세 훼손)
❌ 조정 15거래일 초과 (눌림 아님)
❌ 거래량 급증 동반 하락 (투매)
❌ 악재성 뉴스·공시 동반
   - 유상증자·CB 발행
   - 실적 쇼크 / 가이던스 하향
   - 지배구조 리스크 (횡령·배임)
   - 대형 계약 해지·소송
❌ 섹터 전체 외국인·기관 동반 이탈
❌ 시황 "하락추세" 전환
```

---

## 5. 활용 원칙 (단·중·장 공통 분할 매수 룰)

```
1차 진입 (60%) — 돌파/모멘텀 확인 시 즉시
  → 표준 시점: 신고가 돌파 or 수급 쌍끌이 확인 시

2차 진입 (40%) — 눌림목 구간 도달 시
  → 단기: 5일·20일선 부근 (-5% 이상 조정)
  → 중기: 20일·60일선 부근 (-7% 이상 조정)
  → 장기: 60일·120일선 부근 (-10% 이상 조정)
          or PBR 밴드 하단 접근
```

---

## 6. 눌림 판정 체크리스트 (진입 전 최종 확인)

진입 직전 아래 체크리스트를 **모두 통과**해야 한다:

- [ ] 상승추세 조건 4개 중 **3개 이상** 충족
- [ ] 조정 폭이 시총 구분별 **정상 범위** 내
- [ ] 조정 기간 **15거래일 이내**
- [ ] 조정 구간 거래량 **감소 or 평이**
- [ ] 배제 조건 **모두 해당 없음**
- [ ] 지지선 (이평선/전고점/피보) 중 **1개 이상 근접**
- [ ] 시황 **공격 or 중립** (방어 시 진입 금지)
- [ ] 손절 기준(지지선 이탈 -3~-5%) 사전 설정

---

## 7. 용어 정리 (혼동 방지)

| 용어 | 정의 |
|------|------|
| **눌림목** | 상승 추세 내 일시적 조정 — **매수 타점** |
| **기간 조정** | 가격은 유지되나 시간 경과로 과열 해소 — 눌림 아님 (보유 유지) |
| **데드캣 바운스** | 하락 추세 내 일시 반등 — 눌림과 정반대 (매수 금지) |
| **추세 전환** | 고점·저점이 낮아지기 시작 — 눌림이 아닌 하락 초입 |
| **돌파** | 저항 돌파 후 상승 시작 — 1차 진입 타점 |

## 8. 눌림목 건강도 스코어 (Pullback Health Score, 100점)

> **목적**: 발굴된 후보 종목을 "가장 건강한 눌림목" 순으로 정렬하기 위한 정량 지표.
> **데이터 출처**: 네이버 증권 단독 (`finance.naver.com`, `m.stock.naver.com`)

### 8-1. 배제 조건 (하나라도 해당 시 0점 — 계산 생략)

```
❌ 20일선 < 60일선 (이평선 역배열)
❌ 최근 20일 고점(일봉 기준) 대비 -15% 초과 하락
❌ 조정 16거래일 이상
❌ 조정 구간 거래량 직전 20일 평균比 120% 초과 (투매)
❌ 최근 5거래일 내 악재 공시 (유증·CB·실적 쇼크·횡령·계약 해지)
❌ 당일 시황 = "하락추세" or "방어 심화"
```

### 8-3. 스코어 해석

| 점수 | 등급 | 판정 |
|------|------|------|
| **85~100** | S | 최우선 진입 — 매우 건강한 눌림 |
| **70~84** | A | 진입 추천 — 건강한 눌림 |
| **55~69** | B | 조건부 진입 — 추가 확인 후 |
| **40~54** | C | 관망 — 타점 재형성 대기 |

### 8-4. 정렬 원칙 (리포트용)

```
1순위 정렬: 점수 DESC (85점 → 40점)
2순위 정렬: 주 타점(20일선) 근접도 ASC
3순위 정렬: 시가총액 DESC (대형주 우선)
```
