### 2. 섹션 1 — 단기 유망 종목 (1~2주, 최대 3주)

```markdown
## 📌 단기 유망 종목 (건강한 눌림목 순)

### 🟢 단기 × 대세(+)

| 순위 | 종목 | 코드 | 점수 | 등급 | 조정% | 일수 | 주 타점 | 수급 | 현재가 | 목표가 (+%) | 손절가 (−%) |
|:---:|------|------|:---:|:---:|:----:|:---:|:------:|:---:|-------|------------|------------|
| 1 | 한미반도체 | 042700 | 92 | S | -8.5 | 6 | 20일선 | 동반 | 284,000 | 320,000 (+12.7%) | 269,800 (−5.0%) |
| 2 | … | | | | | | | | | | |

#### 1위: 한미반도체 (042700) — 92점 S등급
- **버킷**: 단기 × 대세(+)
- **눌림목 구조**: 최근 20일 고점(일봉 기준) XXX원에서 -X.X%, X거래일 조정. 20일선 +1.18% 근접, 피보나치 42.3% 되돌림, 거래량 68% (건전)
- **수급**: 외국인+기관 동반 순매수 (최근 20일)
- **투자 근거**: AI 인프라 대세 / HBM 본딩 장비 대장 / 외인·기관 쌍끌이
- **가격 전략**:
  - 현재가 284,000원
  - 목표가 320,000원 (+12.7%)
  - 손절가 269,800원 (-5.0%)
  - R:R = 2.5 : 1
- **리스크**: 단기 급등 피로감, 섹터 이탈 시 비중 축소

### 🔵 단기 × 대세(−)
```

```markdown
- **분할 매수 원칙**: 1차 60% (돌파/모멘텀 확인 시) + 2차 40% (눌림목 도달 시)
```

## 생성 규칙 체크리스트

리포트 확정 직전 아래 모두 통과:

- [ ] 단·중·장 3개 섹션 모두 존재
- [ ] 각 섹션 내 **대세(+) / 대세(−) 서브섹션** 구분
- [ ] 모든 종목 **눌림목 점수 DESC 정렬**
- [ ] 동점 시 **20일선 거리 ASC → 시총 DESC** 2차/3차 정렬 적용
- [ ] 모든 종목에 **현재가·목표가·손절가** 명시
- [ ] **단기 본문**: S/A 등급만, B/C는 대기 목록으로
- [ ] **중기 본문**: S/A/B 등급만, C는 대기 목록으로
- [ ] **장기 본문**: S/A/B/C 모두 포함 가능
- [ ] 대기 목록에 **진입 조건·예상 타점** 명시
- [ ] 배제 종목은 **부록에만** 기재
- [ ] 프론트매터 필수 필드 9개 모두 채움
- [ ] 데이터 출처 네이버 증권 단독 확인
- [ ] 리포트 말미 **각주: 눌림목 건강도 등급 기준** 포함

## 각주 고정 텍스트 (리포트 최하단에 항상 삽입)

```markdown
## 📎 각주: 눌림목 건강도 등급 기준

눌림목 건강도 점수(0~100점)는 **상승추세(A·40pt) + 조정품질(B·35pt) + 지지선근접(C·25pt)** 3개 영역 합산.

| 등급 | 점수 | 의미 | 기간별 리포트 본문 포함 여부 |
|:---:|:---:|------|:---:|
| **S** | 85~100 | 최우선 진입 — 매우 건강한 눌림 | 단기·중기·장기 모두 ✅ |
| **A** | 70~84 | 진입 추천 | 단기·중기·장기 모두 ✅ |
| **B** | 55~69 | 조건부 진입 | 단기 ❌(대기목록) / 중기·장기 ✅ |
| **C** | 40~54 | 관망 — 분할 1차 개념 | 단기·중기 ❌(대기목록) / 장기 ✅ |
| **✕** | <40 또는 배제조건 | 제외 | 부록에만 기재 |

> **배제 조건** (점수 무관하게 즉시 ✕): 이평선 역배열(20일 < 60일) / 최근 20일 고점(일봉 기준) 대비 -15% 초과 / 조정 15거래일 초과 / 거래량 급증(>150%) 동반 하락(투매)
```
