## 명령어 가이드

### "제네시스 하자" (메인 명령어)
**반드시 `포트폴리오관리/.agents/workflows/마스터-제네시스.md` 를 실행한다.**

마스터 워크플로우가 오늘 한국 날짜/요일을 확인하고
아래 워크플로우들을 자동으로 순서대로 호출한다:

```
월~목: 시황분석 → 단기유망종목 → 포트폴리오 리뷰
금요일: 시황분석 → 단기유망종목 → 중기유망종목 → 포트폴리오 리뷰
매달 1일: 위 + 장기유망종목 추가
토/일: 포트폴리오 리뷰만
```

> ⚠️ 개별 폴더의 워크플로우(시황분석/workflows/제네시스.md 등)는
> 마스터 워크플로우 내부에서만 호출된다.
> 사용자가 직접 호출하지 않는다.

## 폴더 구조

```
클라우드코드-포트/
├── 시황분석/          → 일일 시황 리포트 (웹사이트 게시용)
├── 단기유망종목/       → 단기 종목 발굴 리포트
├── 중기유망종목/       → 중기 종목 발굴 리포트 (매주 금요일)
├── 장기유망종목/       → 장기 종목 발굴 리포트 (매달 1일)
├── 종목분석/          → 개별 종목 심층 분석
└── 포트폴리오관리/     → 포트폴리오 대장 + 데일리 리뷰
    ├── .agents/workflows/
    │   ├── 마스터-제네시스.md     ← 메인 진입점
    │   └── 데일리-포트폴리오-리뷰.md
    ├── 포트폴리오-대장.md         ← 보유 종목 + 예산 관리
    └── reviews/                   ← 매일 리뷰 저장
```

## 워크스페이스 격리 원칙 (절대 준수)

> ⛔ 이 워크스페이스(`포트폴리오분석/`)는 독립된 프로젝트다.
> 다른 워크스페이스(눌림목매매, 웹페이지자동화 등)의 파일·폴더를
> 이 레포에 절대 추가하거나 커밋하지 않는다.

- `git add -A` 또는 광범위한 스테이징 시, 현재 워크스페이스 외부 경로가 포함되지 않았는지 반드시 확인한다
- 위 폴더 구조에 없는 최상위 폴더가 생겼다면 즉시 삭제 후 커밋한다
- 커밋 전 `git status`로 변경 파일 목록을 검토한다

## 데이터 수집 주의사항

### ⚠️ 외국인/기관 순매매 상위 테이블 — 칼럼 방향 오류 방지

네이버 증권 **외국인 순매매 상위** / **기관 순매매 상위** 페이지는 두 날짜를 나란히 표시한다:

```
| 종목명(전일) | 수량 | 금액 | 거래량 || 종목명(당일) | 수량 | 금액 | 거래량 |
|  좌측 칼럼  ←─────────────── 전일 ────────────────             당일 ───→  우측 칼럼  |
```

> **좌측 칼럼 = 전일 데이터 / 우측 칼럼 = 당일(오늘) 데이터**
>
> HTML 파싱이나 WebSearch 결과를 해석할 때 **반드시 우측 칼럼(당일)을 사용**한다.
> 좌측 칼럼을 당일 데이터로 오인하는 오류가 반복 발생했으므로 각별히 주의한다.
