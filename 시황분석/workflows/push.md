# 🚀 배포(Push) 워크플로우

본 워크플로우는 생성된 리포트를 지정된 외부 저장소로 자동 배포하는 프로세스를 정의합니다.

## 1. 실행 조건
- 리포트 파일 저장 완료 즉시 자동 실행합니다. 별도 승인 불필요.

## 2. 배포 대상 (Destination)
- **저장소 URL:** `https://github.com/pwman111-debuge/portfolioanalysis`
- **배포 메뉴(경로):** 해당 저장소의 **'시황분석'** 관련 디렉토리 및 메뉴

## 3. 실행 원칙
- 배포 전, 파일의 메타데이터(날짜, 제목, 카테고리) 및 썸네일 경로가 웹사이트 규격에 맞는지 최종 점검합니다.
- 리포트 생성 완료 후 즉시 자동으로 Push합니다.
- 배포 완료 후 결과(Commit Hash 또는 URL)를 즉시 보고합니다.
